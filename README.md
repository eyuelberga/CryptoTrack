Crypto Track helps you track your Cryptocurrencies and get instant email notification whenever they change.

## Features

- Track various market data
- Manage your notifications
- Get latest market data on cryptocurrency


## Development

### Installation

```bash
$ yarn
```

### Running the app

```bash

$ yarn dev
```
